## Run Tests

   Run all tests
   ```
   gradlew clean test
   ```
   
   Run in pacakge
   ```
   gradlew clean test --tests Tests.*
   ```
   
   Run in class all classes in Tests package starting with Selendroid
   ```
   gradlew clean test --tests Tests.Selendroid*
   ```
